# NovaSafe Features

## ✨ Core Features

### 1. Emergency Call Button 🚨
- **Floating button** - Always visible, bottom-right corner
- **One-tap calling** - Direct dial to emergency services
- **Multiple contacts** - 911, Poison Control, Suicide Prevention, Red Cross
- **Pulsing animation** - Draws attention in emergencies
- **Quick 911 button** - Also available in action plan view

### 2. AI-Powered Emergency Analysis
- Real-time analysis using Amazon Nova Pro
- Contextual understanding of emergency situations
- Risk level assessment (LOW/MODERATE/HIGH/CRITICAL)
- Structured action plans with priorities

### 2. Quick Start Templates
- 6 pre-configured emergency scenarios
- One-click analysis for common situations
- Kitchen Fire, Minor Injury, Water Leak, Power Outage, Severe Weather, Car Accident

### 3. Smart Action Plans
- Step-by-step instructions with priorities (CRITICAL/HIGH/MEDIUM)
- Time estimates for each action
- Color-coded risk indicators
- Calming guidance messages

### 4. Share & Export
- **Print Action Plan** - Print-friendly format for physical reference
- **Copy to Clipboard** - Share via text/email
- Formatted for easy reading

### 5. Analysis History
- Saves last 3 emergency analyses
- Quick access to previous plans
- Timestamp tracking
- Risk level indicators

### 6. Response Time Tracking
- Shows how fast Nova analyzed the situation
- Transparency in AI processing
- Performance metrics

### 7. Emergency Resources
- Contextual emergency contact numbers
- Support service recommendations
- Local resource suggestions
